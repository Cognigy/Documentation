# TTS and STT Vendors

The table below shows the TTS (Text-to-Speech) and STT (Speech-to-Text) providers supported by Cognigy Voice Gateway.

| Provider   | TTS | STT |
|------------|-----|-----|
| Microsoft  | +   | +   |
| AWS        | +   | +   |
| Google     | +   | +   |
| Nuance     | +   | +   |
| Soniox     | -   | +   |
| Custom[^*] | -   | -   |


[^*]: The **Custom** option is temporarily not supported.
