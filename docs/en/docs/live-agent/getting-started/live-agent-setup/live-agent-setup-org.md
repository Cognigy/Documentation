# Set up Live Agent for your Organization

Before you can begin using Live Agent, you will need to set up Live Agent for your Organization.

## First Steps

1. Log in to your Cognigy.AI interface, and click the **User Menu**. You will find the icon in the top right corner of your user interface.
2. Click the **Organisation Details** menu item. You can see the section for Live Agent organization. 
3. Find the **Set up Live Agent** button, click it to set up the Live Agent Account for your Organization.

## Troubleshooting

If the **Set up Live Agent** button is disabled, it could be for the following reasons:

1. You do not have the Cognigy Live Agent feature enabled.
2. You do not have permissions to create a Live Agent account.
3. There is already a Live Agent account created for your Organization.

!!! note
    The text above the button can help you find why the button is disabled for you.
