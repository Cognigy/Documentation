# Agents

An Agent is a member of your Customer Support team. Agents will be able to view and reply to messages from your users.

Click on the sidebar **Settings** button on the Live Agent Dashboard.

<figure>
<img class="image-center" src="{{config.site_url}}live-agent/images/LA-dashboard-settings.png" width="100%" alt="Live Agent Dashboard Home Screen">
  <figcaption>Live Agent Dashboard</figcaption>
</figure>

The section shown below will open. Your name will be listed by default as Administrator. An Administrator will have access to all Live Agent features enabled for your account, including settings, along with all of a normal Agents' privileges.

<figure>
<img class="image-center" src="{{config.site_url}}live-agent/images/LA-agents-home.png" width="100%" alt="Live Agent Agents Administration Screen">
  <figcaption>Agents Administration Screen</figcaption>
</figure>

Click on the Add Agent button on the top right-hand side. A popup window will open in which you can enter the details of the Agent, such as their name, role and email address. 

<figure>
<img class="image-center" src="{{config.site_url}}live-agent/images/LA-agents-add-agent-to-team.png" width="100%" alt="Live Agent Add Agent to Team">
  <figcaption>Add Agent to Team</figcaption>
</figure>


| Field Name           | Value                                    | Remarks                                                                                                                                             |
|----------------------|------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------|
| Agent Name           | Enter Name of the Agent                  | e.g. Jane Doe                                                                                                                                        |
| Role                 | Enter the Role - Agent or  Administrator | Agents can only access Inboxes and  Conversations. Administrators have all privileges of Agents and have access to all features of Live Agent |
| Your e-mail  address | The Agent Email    |                                                                                                                                                     |

Click on the **Add Agent** button. An "Agent Added Successfully" message will be displayed, and the New Agent will be listed on the page.

If you have entered the details incorrectly, this can be corrected later using the Edit option in the **Agents** screen.

<figure>
<img class="image-center" src="{{config.site_url}}live-agent/images/LA-agents-edit-agent.png" width="100%" alt="Live Agent Edit Agent Dialog">
  <figcaption>Edit Agent Dialog</figcaption>
</figure>

In case the particular Agent no longer works with you, you can go ahead and use the delete option to delete the Agent from the dashboard.

<figure>
<img class="image-center" src="{{config.site_url}}live-agent/images/LA-agents-delete-agent.png" width="100%" alt="Live Agent Delete Agent Dialog">
  <figcaption>Delete Agent Dialog</figcaption>
</figure>
