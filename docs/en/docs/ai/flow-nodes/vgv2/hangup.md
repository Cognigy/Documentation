## Description
<div class="divider"></div>
The Hang Up Node will instruct Voice Gateway to end the call.

| Parameter | Type | Description |
| ----------- | ----------- | ----------- |
| Reason for hang up | Text | The reason for hanging up, will be shown in logs |
