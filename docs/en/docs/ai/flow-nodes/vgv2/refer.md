## Description
<div class="divider"></div>
The Refer Node let's you pass an active call to a tel or sip target.

| Parameter | Type | Description |
| ----------- | ----------- | ----------- |
| Refer Destination | Text | E.164 syntax or a SIP URI are supported. |
