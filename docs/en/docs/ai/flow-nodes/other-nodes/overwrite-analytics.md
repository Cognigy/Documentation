## Description
<div class="divider"></div>
Gives the ability to selectively overwrite the [Analytics data]({{config.site_url}}ai/flow-nodes/code/analytics-data/)  which will be stored into [Analytics]({{config.site_url}}ai/resources/analytics/);

Please see [Analytics data]({{config.site_url}}ai/flow-nodes/code/analytics-data/) for the fields which can be overwritten.

???+ warning "Selective Overwrite"
    Blank fields will remain "as-is", meaning they will contain the data which Cognigy.AI would store by default.
