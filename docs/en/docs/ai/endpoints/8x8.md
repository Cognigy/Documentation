## Channel Specific Settings

### 8x8 Setup

Base URL - is the URL of the 8x8 Server.

API Key - is the header value required for authenticating API calls to the 8x8 system.

8x8-Tenant - is mandatory if a customer has multiple CC tenants in their organization and requires providing the Tenant ID.
