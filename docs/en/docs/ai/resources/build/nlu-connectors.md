# NLU Connectors

Cognigy.AI features built-in support for a number of third-party NLU engines. These are: 

  * **Google Dialogflow**
  * **IBM Watson Assistant**
  * **Amazon Alexa**
  * **Amazon Lex**
  * **Microsoft LUIS**
  * **Code (custom)**
